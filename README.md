# React-Native-Expo
### This is a template to start building your React Native with Expo application for IOS/Andoid/Web application.

### Scripts
```"scripts": {
    "start": "expo start --tunnel",
    "android": "expo start --android",
    "ios": "expo start --ios",
    "web": "expo start --web",

```
